    # 数组基础知识整理

    # 1.概念

    • 数组是对象的一种特殊形式
    • 数组的length属性是不可枚举的属性
    • 循环数组的两种方式 for for...in 只能遍历可枚举属性

    # 2.数组的常用方法

    # 2.1 push/unshift 有参数

    • 向数组末尾/开头追加内容
    • 改变原数组
    • 返回新数组的长度

    # 2.2 pop/shift 无参数

    • 删除数组末尾/开始项
    • 改变原数组
    • 返回被删除的项

    # 2.2 splice 有参数(start[, deleteCount[, item1[, item2[, ...]]]])

    • 功能：通过删除或替换现有元素或者原地添加新的元素来修改数组
    • 原数组：改变原数组
    • 返回：以数组形式返回被修改的内容
    • 注：start 为负数，表示反方向开始
